# Как запустить Kaspi на ПК

Напрямую через file:// открывать НЕЛЬЗЯ — браузер блокирует камеру и манифест.
Нужен локальный сервер.

## Способ 1 — Двойной клик (Windows)
1. Дважды кликни на файл **ЗАПУСТИТЬ.bat**
2. Браузер откроется автоматически на http://localhost:8080

## Способ 2 — Python вручную
Открой терминал в папке проекта и выполни:
```
python -m http.server 8080
```
Затем открой: http://localhost:8080

## Способ 3 — Node.js
```
npx serve -l 8080 .
```

## На iPhone (PWA)
Открой сайт через HTTPS-хостинг (GitHub Pages, Vercel и т.д.),
затем Safari → Поделиться → На экран «Домой».
